## Features

### Colors

- Red: indicates both the block-light-level and sky-light-level are 0.

- Yellow: indicates only the block-light-level is 0.

- Green: indicates both are 1 or higher.

### Debug mode

- Press F3 to show both block-light-level and sky-light-level.
